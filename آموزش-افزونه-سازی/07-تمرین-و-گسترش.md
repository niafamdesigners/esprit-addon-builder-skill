# بخش ۷ — تمرین و گسترش

## تمرین ۱: افزودن فیلد تصویر به افزونه پیوندها

**هدف:** افزودن قابلیت تصویر (favicon / لوگو) به هر لینک

### فیلد جدید که باید اضافه کنید:

```json
{
  "friendlyname": "تصویر (Favicon)",
  "fieldname": "favicon",
  "fieldtype": "file",
  "length": 255,
  "filewidth": 64,
  "fileheight": 64,
  "direction": "ltr",
  "deleted": 0,
  "ord": 6,
  "showonlist": 1,
  "subform": 0,
  "search": 0,
  "required": 0,
  "friendlyname_en": "Favicon"
}
```

### تغییر در کوئری:
```sql
SELECT
    ISNULL(title,'') AS title,
    ISNULL(description,'') AS description,
    ISNULL(link_url,'#') AS link_url,
    ISNULL(category,'') AS category,
    ISNULL(target,'_self') AS target,
    ISNULL(favicon,'') AS favicon      ← فیلد جدید
FROM links
WHERE deleted = 0 AND siteid = [system:site-id]
ORDER BY CAST(ordlist AS INT) ASC
```

### تغییر در repeathtml:
```html
<div class="col-md-6 col-lg-4 mb-3">
  <div class="card h-100 border-0 shadow-sm">
    <div class="card-body d-flex align-items-start gap-3">
      <img src="[query-result-fileurl:favicon]"    ← اضافه شد
           width="32" height="32"
           alt=""
           onerror="this.style.display='none'">
      <div>
        <h5 class="card-title mb-1">
          <a href="[query-result:link_url]" target="[query-result:target]">
            [query-result:title]
          </a>
        </h5>
        <p class="card-text text-muted small">[query-result:description]</p>
        <span class="badge rounded-pill bg-light text-dark">[query-result:category]</span>
      </div>
    </div>
  </div>
</div>
```

## تمرین ۲: افزونه سوالات متداول (FAQ)

**هدف:** ساخت افزونه برای سوال و جواب‌های متداول

### فیلدها:

| fieldname | نوع | اجباری |
|-----------|-----|--------|
| question | text | بله |
| answer | textarea | بله |
| category | text | خیر |
| ordlist | selectbox | بله |

### کوئری:
```sql
SELECT
    ISNULL(question,'') AS question,
    ISNULL(answer,'') AS answer,
    ISNULL(category,'') AS category
FROM faq_items
WHERE deleted = 0 AND siteid = [system:site-id]
ORDER BY CAST(ordlist AS INT) ASC
```

### قالب HTML (آکاردئون):
```html
<!-- starthtml -->
<div class="accordion" id="faqAccordion">

<!-- repeathtml -->
  <div class="accordion-item">
    <h2 class="accordion-header">
      <button class="accordion-button collapsed" type="button" 
              data-bs-toggle="collapse" 
              data-bs-target="#faq-[query-result:id]">
        [query-result:question]
      </button>
    </h2>
    <div id="faq-[query-result:id]" class="accordion-collapse collapse">
      <div class="accordion-body">
        [query-result:answer]
      </div>
    </div>
  </div>

<!-- endhtml -->
</div>
```

## تمرین ۳: افزونه اعضای تیم

**هدف:** نمایش اعضای تیم با تصویر و اطلاعات تماس

### فیلدها:

| fieldname | نوع | اجباری |
|-----------|-----|--------|
| name | text | بله |
| position | text | بله |
| bio | textarea | خیر |
| photo | image | خیر |
| textinput | email | خیر |
| textinput | phone | خیر |
| linkedin_url | textinput | خیر |
| ordlist | selectbox | بله |

### کوئری:
```sql
SELECT
    ISNULL(name,'') AS name,
    ISNULL(position,'') AS position,
    ISNULL(bio,'') AS bio,
    ISNULL(photo,'') AS photo,
    ISNULL(email,'') AS email,
    ISNULL(phone,'') AS phone,
    ISNULL(linkedin_url,'#') AS linkedin_url
FROM team_members
WHERE deleted = 0 AND siteid = [system:site-id]
ORDER BY CAST(ordlist AS INT) ASC
```

## نکات مهم برای گسترش

### ۱. اضافه کردن فیلتر بر اساس دسته‌بندی

اگر می‌خواهید کوئری فقط یک دسته را نمایش دهد:

```sql
WHERE deleted = 0 
  AND siteid = [system:site-id] 
  AND category = 'توسعه'
```

### ۲. محدود کردن تعداد نتایج

برای نمایش فقط ۶ آیتم اول:
```sql
SELECT TOP 6
    ISNULL(title,'') AS title,
    ...
FROM links
WHERE deleted = 0 AND siteid = [system:site-id]
ORDER BY CAST(ordlist AS INT) ASC
```

### ۳. مرتب‌سازی بر اساس زمان

برای نمایش جدیدترین آیتم‌ها اول:
```sql
ORDER BY createtime DESC
```

### ۴. چند کوئری برای یک افزونه

می‌توانید چند کوئری با نام‌های مختلف برای یک افزونه تعریف کنید:

```json
"queries": [
  {
    "queryname": "list_all_links",
    "query": "SELECT ... FROM links WHERE ..."
  },
  {
    "queryname": "top6_links",
    "query": "SELECT TOP 6 ... FROM links WHERE ..."
  },
  {
    "queryname": "links_by_dev",
    "query": "SELECT ... FROM links WHERE ... AND category = 'توسعه'"
  }
]
```

## جدول مرجع سریع — همه چیز در یک نگاه

### انواع فیلد:
```
text       → متن کوتاه (۲۵۵ کاراکتر) — direction: rtl
textarea   → متن بلند — direction: rtl
url        → آدرس لینک — direction: ltr
email      → آدرس ایمیل — direction: ltr
phone      → شماره تلفن — direction: ltr
number     → عدد — direction: ltr
image      → تصویر — direction: ltr
file       → فایل — direction: ltr
selectbox  → لیست کشویی ثابت
checkbox   → بله/خیر — direction: ltr
date       → تاریخ — direction: ltr
```

### قالب کوئری:
```sql
SELECT
    ISNULL(field1,'') AS field1,
    ISNULL(field2,'') AS field2,
    ISNULL(url_field,'#') AS url_field
FROM tablename
WHERE deleted = 0 AND siteid = [system:site-id]
ORDER BY CAST(ordlist AS INT) ASC
```

### الگوهای HTML:
```
[query-result:fieldname]          → برای متن، لینک، عدد
[query-result-fileurl:fieldname]  → برای تصویر و فایل
```

## خلاصه مسیر ساخت افزونه

```
۱. طراحی ← چه فیلدهایی نیاز داریم؟
        ↓
۲. تعریف جدول ← tablename، مجوزها
        ↓
۳. تعریف فیلدها ← fieldname، fieldtype، direction، ord
        ↓
۴. نوشتن کوئری ← SELECT ... FROM ... WHERE ... ORDER BY
        ↓
۵. طراحی HTML ← starthtml + repeathtml + endhtml
        ↓
۶. ترکیب در JSON ← addon + table + fields + queries
        ↓
۷. ثبت در سیستم ← افزونه آماده!
```
